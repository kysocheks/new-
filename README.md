# Сайт-лендинг «Специалист по информационной безопасности»
  
Сайт объясняет профессию **«Специалист по информационной безопасности (защита инфраструктуры и конечных точек)»** простым и доступным языком.

---

## 🧩 Структура сайта (все блоки по ТЗ)

| Блок | Описание |
|------|----------|
| **Hero** | Название профессии, слоган, кнопки «Узнать больше» и «Пройти тест» |
| **«Кто это простыми словами»** | Понятное объяснение профессии для первокурсника |
| **Рабочие задачи** | 7 карточек с иконками (анализ угроз, защита, мониторинг и т.д.) |
| **Hard skills и Soft skills** | Две колонки навыков (технические и личностные) |
| **Как начать карьеру студенту** | Пошаговый план: основы → практика → опыт → рост |
| **Один день из жизни** | Таймлайн с 6 временными слотами (09:00–17:00) |
| **Почему роль важна для ЭРТХ** | Текст + статистика (500+ серверов, 1000+ атак) |
| **Интерактивный тест** | 4 вопроса по ИБ с подсчётом результатов и обратной связью |

---

## 🛠 Использованные технологии

- **React 18** + **Vite** – быстрая разработка и горячая перезагрузка
- **Framer Motion** – плавные анимации (появление при скролле, hover-эффекты)
- **UI/UX Pro Max** – готовая дизайн-система (цвета, шрифты, отступы)
- **21st.dev** – готовые компоненты (Hero, карточки, таймлайн)
- **Vanta.js** – интерактивный фон «Кибер-сеть» с реакцией на мышь
- **Git + GitHub** – система контроля версий
- **Vercel** – автоматический деплой из ветки `main`

---

## 🤖 Основные промпты, использованные для генерации

1. *«Создай React-лендинг (Vite) для профессии "Специалист по информационной безопасности" с тёмной темой, зелёным акцентом (#00FF88), Framer Motion и структурой: Hero, О профессии, Задачи, Навыки, Карьера, День из жизни, Важность, Тест.»*

2. *«Добавь интерактивный тест из 4 вопросов с подсчётом результатов, прогресс-баром и мгновенной обратной связью (подсветка правильного/неправильного ответа).»*

3. *«Установи Vanta.js (эффект сети) на фон, отключай на мобильных. Добавь кастомный эластичный курсор с эффектом пружины (useSpring).»*

4. *«Оптимизируй адаптивность, исправь мигание текста (замени лишние motion-компоненты на обычные, добавь key в списки).»*

---

## ✍️ Ручные правки после ИИ 

1. **Тексты** – заменили шаблонные формулировки на живые, добавили конкретные названия инструментов.
2. **Цветовая гамма** – изменили акцент с синего на зелёный (#00FF88), чтобы соответствовать теме ИБ и выглядеть более технологично.
3. **Статистика** – вместо общих фраз вставили реальные цифры (например, «500+ защищённых серверов», «1000+ предотвращённых атак»).
4. **Исправление багов** – убрали мигание заголовков, заменив `motion.h1` на обычные `h1`, и добавили `key` в списки карточек.
